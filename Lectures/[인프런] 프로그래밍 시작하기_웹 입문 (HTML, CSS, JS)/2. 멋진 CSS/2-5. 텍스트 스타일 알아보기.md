### 강의 내용

- 문장, 문단과 관련된 스타일
  - text-align
    - 정렬 방식.
    - left, right, center, justify 중 선택
    - justify는 균등하게 정렬해주는 것. 글자 간격이 넓어질 수 있음.
  - text-indent
    - px 등을 이용해서 첫 글자 들여쓰기 수준 정의.
  - text-decoration
    - 글자 장식.
    - underline: 밑줄
    - overline: 윗줄
    - line-through: 취소선
