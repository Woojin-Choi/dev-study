### 강의 내용

- 배열형(Array Type)

  - 인덱스 기반으로 값을 순서대로 삽입할 수 있는 특별한 객체 형태
  - 인덱스: 위치. 0부터 시작하는 정수형 값.

    ```js
    var array = [0, 10, 20];

    // 인덱스 접근
    array[0]; // 0
    array[2]; // 20
    ```

- 배열 메소드
  - 속성
    - length: 배열의 크기 리턴
  - 함수
    - push(): 마지막 인덱스로 새로운 값 추가
    - unshift(): 0 인덱스로 새로운 값 추가
    - pop(): 마지막 인덱스 값 리턴하면서 제거
    - shift(): 0 인덱스 값 리턴하면서 제거
